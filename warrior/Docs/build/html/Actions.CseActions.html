execute_cse_script(system_name, cse_logfile_path='default')[source]¶

Executes CSE’s run script (cse’s wrapper script) on the cse view. 1. Connects to the Server IP 2. Sets the view 3. Go to the cse run script location 4. Execute the cse run script on the server 5. Send the log file from the server to Warrior logsdir 6. Checks the log for the run result

Arguments:
  1. system_name(string) = name of the system in the input datafile
  2. cse_logfile_path = full path to the cse logfile, default=’default’
Returns:
  1. status(bool) = True or False
